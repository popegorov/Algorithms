Дана строка S, состоящая из строчных латинских букв.
Рассмотрим две её подстроки длины L, первая из которых начинается в индексе A, а вторая — в индексе B.
Научитесь эффективно определять, какая из этих подстрок лексикографически меньше (то есть следует раньше в алфавитном порядке).
Формат ввода

Первая строка содержит последовательность S (1 ≤ |S| ≤ 2 ⋅ 105), состоящую из строчных латинских букв.
Вторая строка содержит целое число Q (1 ≤ Q ≤ 2 ⋅ 105) — количество запросов.
Следующие Q строк описывают запросы. Каждая из них содержит целые числа Li, Ai и Bi (1 ≤ Li ≤ |S|, 0 ≤ Ai, Bi ≤ (|S| - Li)) — соответственно длину подстрок и индексы начала каждой из них.
Формат вывода

Для каждого из запросов выведите -1, если первая подстрока лексикографически меньше второй, 0, если подстроки равны, или 1, если первая подстрока лексикографически больше второй.
