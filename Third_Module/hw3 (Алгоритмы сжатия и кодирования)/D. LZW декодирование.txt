Выполните распаковку сжатого текстового сообщения по алгоритму LZW.
Словарь кодирующих фрагментов должен изначально содержать 128 значений (0 – 127) для отдельных ASCII-символов с соответствующими кодами.
Формат ввода

Первая строка содержит целое число K (1 ≤ K ≤ 104) — количество фрагментов, кодирующих сообщение в соответствии с алгоритмом LZW.
Вторая строка содержит K целых чисел Ci (0 ≤ Ci ≤ 104) — кодирующие фрагменты в десятичной записи.
Формат вывода

Выведите одну строку — исходное сообщение.
