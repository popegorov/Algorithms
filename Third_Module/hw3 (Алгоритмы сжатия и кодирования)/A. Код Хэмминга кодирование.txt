Дана битовая строка, биты нумеруются слева направо.
Постройте для данной строки код Хэмминга. Количество дополнительных контрольных битов должно быть минимально возможным.
Формат ввода

Ввод содержит строку S (1 ≤ |S| ≤ 105), состоящую из символов 0 и 1, — исходное сообщение для кодирования.
Формат вывода

Выведите одну строку — код Хэмминга для сообщения.
